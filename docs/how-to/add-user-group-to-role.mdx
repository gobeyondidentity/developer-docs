## Admin Console
1. In the Admin Console, under Authorization, click **Roles**  
2. Click the role you wish to add a user or group to (or otherwise [create a new role](/docs/next/create-role))  
3. If you wish to add a user, on the **Identities** tab click **Add identity**   
4. If you wish to add a group, click the **Groups** tab and click **Add group**    
5. Select the identity or group you wish to add and click **Add to role**
