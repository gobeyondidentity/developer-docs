## API

Realms can also be created by API. Before making any API calls you'll want to generate an API access token. Check out [API Tokens](/docs/create-api-token#admin-console) for help creating an access token. Once you have an access token, make a request to the `/realms` endpoint below.

<MultiLanguageCodeBlock
curl='curl "https://api-$(REGION).beyondidentity.com/v1/tenants/$(TENANT_ID)/realms" \
-X POST \
-H "Authorization: Bearer $(TOKEN)" \
-H "Content-Type: application/json" \
-d "{\"realms\":{\"display_name\":\"$(REALM_NAME)\"}}"'
title="/realms"
/>
