An Authenticator Config tells us how and where to invoke passkey creation, and authentication flows for your native/mobile or web application. 

Each application is tied to a single authenticator type, which is configured in Authenticator Config. This article will help you understand each authenticator type option and how they impact Beyond Identity passkey flows.
